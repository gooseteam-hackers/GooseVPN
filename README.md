# 🪿 GooseVPN

## Дисклеймер
Данный проект создан исключительно в образовательных целях. Родину люблю, государство уважаю, но VPN все равно нужен ;)

## В чем прикол
Это автоматический сборщик, который каждые 24 часа собирает новый конфиг похожий на платные VPN, отсортированно и красиво.

## Типы конфигов внтури
- 🔮 WiFi - тут название говорит само за себя. Тут все стабильно и открывает все двери
- 🤍 wLTE - конфигурации, ну, кхм-кхм... для мобильных данных 😉 тут главное просто войти в интернет
- 🔥 WARP - есть стабильный конфиг и ночной конфиг. Стабильный не обновляется (потом может обновлю, если надо) и ночной обновляется каждый день
- 🪿 Резервы - не вошли в основной топ, но существуют на пожарный случай

## Начать работу

### Рекомендуемые клиенты
#### Hiddify
Лучший из всех.
➕ Плюсы:
- Sing-box - эталон скорости, благодаря чему он быстрый и легкий
- Интерфейс простой и красивый
- Поддержка ссылок с подписками
- Опен-сорс, можно доверять
- Много настроек

➖ Минусы:
- Новички могут не сразу понять где сменить конфиг. По умолчанию выбирает наилучший, но если включить VPN и нажать снизу на конфиг, можно выбрать
- Размер немного большеват
- Недоступен в App Store в русском регионе

#### Happ
➕ Плюсы:
- Прост в использовании
- Популярен в широких массах
- Доступен в русском App Store

➖ Минусы:
- Не опен-сорс, нельщя полностью доверять
- Обфускация приложения похожа на ту что есть в MAX, что вызывает вопросы, ведь он еще и делается в России

#### Kairing
➕ Плюсы:
- Лучшее ядро sing-box
- Доступен в русском App Store
- Высокая кастомизабельность UI
- Опен-сорс, можно доверять

➖ Минусы:
- Интерфейс может быть сложен новичкам
- Нужно время для настройки

#### v2RayTun
➕ Плюсы:
- Есть в русском App Store
- Хорошая скорость (все таки sing-box будет в разы быстрее)
- Простой UI

➖ Минусы:
- Не опен-сорс, нельзя полностью доверять
- Может быть медленным

### Какую версию мне выбрать
Есть две версии - обычная и Plus. 
- Обычная версия содержит по 3 конфига на каждый основной тип (wLTE, WiFi, Резерв). Если вы обычный человек, можете попробовать эту версию, там меньше конфигов, значит легче перебрать

Обычная подписка:
```
https://raw.githubusercontent.com/gooseteam-hackers/GooseVPN/refs/heads/main/configs/balanced.txt
```
- Plus версия содержит все из обычной версии, но теперь уже по 4 конфига каждого типа, а также есть WARPы и более красивое название 😉. Выбирайте ее если вы хотите лишние конфиги, WARP и красивенькое Plus в названии, а также получать новые фишки (нестабильные)

Plus подписка:
```
https://raw.githubusercontent.com/gooseteam-hackers/GooseVPN/refs/heads/main/configs/plus.txt
```

### Установка
Все предельно просто.

1. Установите любой клиент с поддержкой v2ray/xray и подписок в ссылках
2. Импортируем ссылку выше (с нужной версией)
3. Подключаем и пользуемся!
