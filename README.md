## 📋 О проекте

GooseVPN — это автоматический парсер, который собирает бесплатные VLESS-конфигурации из открытых источников, фильтрует небезопасные подключения, тестирует скорость и создаёт готовые подписки для VPN-клиентов.

### ⚡ Возможности

- 🔍 **Автоматический сбор** — 20+ источников конфигураций
- 🛡️ **Фильтрация insecure** — отбрасывает конфиги с `allowInsecure=true`
- 🌍 **Geo-определение** — определяет страну сервера по IP
- ⚡ **Ping-тесты** — воронка тестов для выбора лучших серверов
- 🧹 **Дедупликация** — удаляет дубликаты по IP:PORT
- 📝 **Готовые подписки** — совместимость с популярными клиентами

## 🚀 Быстрый старт

### 1. Выберите VPN-клиент

| Клиент | Платформы | Плюсы | Минусы |
|--------|-----------|-------|--------|
| **Hiddify** ⭐ | Windows, macOS, Linux, Android, iOS | Sing-box (быстрый), красивый UI, open-source | Нет в App Store RU |
| **Karing** | Windows, macOS, Android, iOS | Лучшее ядро sing-box, кастомизация | Сложный для новичков |
| **v2RayTun** | Android, iOS | Простой, хорошая скорость | Не open-source |
| **Happ** | Android, iOS | Популярен, прост | Не open-source |

### 2. Импортируйте подписку

#### 📦 Balanced (обычная версия)
Меньше конфигов, легче перебрать. Подходит для большинства пользователей.

```
https://raw.githubusercontent.com/gooseteam-hackers/GooseVPN/refs/heads/main/configs/balanced.txt
```

**Состав:** 6 wLTE + 6 WiFi + 4 резерва + 2 WARP = **18 конфигов**

#### 💎 Plus (расширенная версия)
Больше конфигов, все типы подключений, включая WARP.

```
https://raw.githubusercontent.com/gooseteam-hackers/GooseVPN/refs/heads/main/configs/plus.txt
```

**Состав:** 10 wLTE + 10 WiFi + 5 резервов + 2 WARP = **27 конфигов**

### 3. Настройка в Hiddify

1. Откройте Hiddify
2. Нажмите **«+»** → **«Импорт из ссылки»**
3. Вставьте ссылку на подписку
4. Нажмите **«Сохранить»**
5. Выберите лучший сервер (или оставьте автовыбор)
6. Подключайтесь! 🎉

## 📊 Типы конфигураций

| Тип | Значок | Описание |
|-----|--------|----------|
| **wLTE** | 🤍 | Для мобильных сетей. Работает через LTE/5G |
| **WiFi** | 🔮 | Стабильные конфигурации для домашнего интернета |
| **WARP** | 🔥 | Cloudflare WARP (стабильный + ночной) |
| **Резерв** | 🪿 | Запасные конфиги на случай проблем |

## 🛠️ Запуск локально

### Требования

- Python 3.8+
- pip

### Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/gooseteam-hackers/GooseVPN.git
cd GooseVPN

# Установите зависимости
pip install -r requirements.txt

# Запустите парсер
python main.py
```

### Аргументы командной строки

```bash
# Полный запуск с ping-тестами (~5-10 минут)
python main.py

# Быстрый запуск без тестов (~30 секунд)
python main.py --skip-funnel

# Только Balanced версия
python main.py --only-balanced

# Только Plus версия
python main.py --only-plus

# Отладочная информация
python main.py --debug
```

| Аргумент | Описание |
|----------|----------|
| `--skip-funnel` | Пропустить ping-тесты |
| `--only-balanced` | Сгенерировать только balanced.txt |
| `--only-plus` | Сгенерировать только plus.txt |
| `--debug` | Показать отладочную информацию |
| `--wlte-sources FILE` | Файл с источниками wLTE |
| `--wifi-sources FILE` | Файл с источниками WiFi |
| `--geo FILE` | Путь к GeoLite2-Country.mmdb |

### Структура проекта

```
GooseVPN/

```

## 🔧 Как это работает

1. **Сбор** — загрузка конфигураций из 20+ источников
2. **Обработка** — URL-decode, HTML-unescape, base64-декодирование
3. **Фильтрация** — удаление insecure конфигов (`allowInsecure=true`)
4. **GeoIP** — определение страны сервера
5. **Ping-тесты** — 3-этапная воронка для выбора лучших
6. **Генерация** — создание подписок с красивыми именами

## 📈 Статистика

Обычно парсер собирает:
- **~10,000** wLTE конфигураций
- **~8,000** WiFi конфигураций
- **~7,000** отфильтровывается (insecure)
- **~48** проходит в финальную выборку
- **18-27** попадает в подписки
